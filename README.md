# RiffWave – Theme Switcher UI

Proyecto de demostración de sistemas visuales adaptativos usando CSS Variables.

## Funcionalidades
- Tema Claro / Oscuro
- Cambio instantáneo global
- Paleta accesible (WCAG AA/AAA)
- Componentes reales: navbar, cards, botones, inputs

## Tecnologías
- HTML5
- CSS Variables
- JavaScript Vanilla
